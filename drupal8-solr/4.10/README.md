# SOLR

```Dockerfile
FROM quay.io/inviqa_images/drupal8-solr:4.10
```

## How to build
```bash
docker build --pull --tag quay.io/inviqa_images/drupal8-solr:4.10 --rm .
docker push
```

## How to use

As this is based off of a semi-official solr image, please see their README, here:
